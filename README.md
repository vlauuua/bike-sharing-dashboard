# 🚲 Bike Sharing Dashboard ✨

Project ini bertujuan untuk menganalisis pola penyewaan sepeda menggunakan dataset Bike Sharing (2011–2012).  
Analisis dilakukan untuk mengetahui pengaruh faktor seperti cuaca, hari, dan bulan terhadap jumlah penyewaan sepeda.

Hasil analisis divisualisasikan dalam bentuk dashboard interaktif menggunakan Streamlit.

## 📊 Fitur Dashboard
- 📌 Key Metrics (Total, Rata-rata, Maksimum penyewaan)
- 🌤️ Analisis penyewaan berdasarkan kondisi cuaca
- 📅 Analisis penyewaan berdasarkan hari (weekday)
- 📈 Tren penyewaan bulanan
- 🎛️ Filter interaktif berdasarkan kondisi cuaca

## ⚙️ Setup Environment

### 🔹 Setup Environment - Anaconda
conda create --name main-ds python=3.9
conda activate main-ds
pip install -r requirements.txt

---

### 🔹 Setup Environment - Shell/Terminal
mkdir proyek_analisis_data
cd proyek_analisis_data
pip install pipenv
pipenv install
pipenv shell
pip install -r requirements.txt
